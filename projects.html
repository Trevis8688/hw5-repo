Relevant Coursework

  • CSE 100: Advanced Data Structures

    High-performance data structures and supporting algorithms. Topics include the use and implementation of data structures like balanced and unbalanced trees, graphs, priority queues, and hash tables, as well as memory management, pointers, and recursion. The course emphasizes both theoretical and practical performance analysis, using C++ and the Standard Template Library (STL).

  • CSE 101: Design and Analysis of Algorithms

    Design and analysis of efficient algorithms with an emphasis on non-numerical algorithms such as sorting, searching, pattern matching, and graph and network algorithms. The course covers measuring the complexity of algorithms, including time and storage, and discusses NP-complete problems.

  • CSE 132A: Introduction to Database Systems

    Basic concepts of databases, including data modeling, relational databases, query languages, optimization, dependencies, schema design, and concurrency control. The course provides exposure to one or several commercial database systems and may cover advanced topics such as deductive and object-oriented databases, time permitting.

  • CSE 110: Software Engineering

    An introduction to the principles of software engineering, including software development life cycles, requirements analysis, design methodologies, testing strategies, and project management. The course emphasizes practical experience through team-based projects, fostering skills in collaboration, communication, and the application of software engineering practices.
